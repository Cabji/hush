Hush Searches Import Utility

Introduction

This feature of Hush allows you to semi-automatically import media titles from an external source into Hush's passive search list. It is ideal for finding upcoming releases and having them simply added into Hush. To access the Searches Import windows, do the following: Hush Main window, click Searches button, click Import Searches... button

URL - The URL is the external page that Hush will look at and analyze to find media titles.
Import Type - The dropdown box will be a selection of external sources that you can import titles from. Currently the only option is IMDB's Lists.
LAUNCH button - LAUNCHes the import.

To obtain a URL for the import utility (opposite click and select "Copy Link Location"):

  1. Go to IMDB's Advanced Search Page
  2. Build a search using the options on that page
  3. Click the Search button at the bottom
  4. When the page loads you will see all the results returned. Copy the URL from the URL bar of your browser.
  5. Paste this URL into Hush's Import utility

Optionally, you can use some of these premade searches:
  • Upcoming Popular English Feature Films within the Next Year (from May 2018 to May 2019, current at time of writing)
  • Most Popular English Comedy Feature Films since 2017 onward
  • IMDB's Top 100 of All Time English Feature Films
Note that the utility will pull the results from the loaded page only, not the entire list of results returned by IMDB's search. If you want results from additional pages load the page and paste that URL into the import utility. Be aware that IMDB's searches can return thousands of results so you should build a search you like first on their website, checking exactly what comes up and then import it. You can also control the number of titles shown per page via the IMDB's advanced search options.

The importer will also pull title names from IMDB's premade Lists. You can find lists by googling for them. Lists cna also be refined with additional conditions to narrow them down to whatever suits your needs. The handiest list I found was for new and recently release bluray or dvd titles, which can be refined down further to suit your taste like the following links show:

  • New and Recent DVD/Blu-ray Releases
  • New and Recent DVD/Blu-ray Releases, Feature Films only, Released 2018 onward, 2000 User Votes minimum
By refining titles to have at least 2000 user votes you filter out more of the lesser known titles. You could inverse that refinement if you want lesser known titles.
